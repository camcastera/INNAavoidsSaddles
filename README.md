# Code for Inertial Newton Algorithms Avoiding Strict Saddle Points

This repository contains the experiments and figures for the paper [*Inertial Newton Algorithms Avoiding Strict Saddle Points*](https://arxiv.org/abs/2111.04596) by C. Castera. Experiments are were made in Python 3 and require numpy and matplotlib packages.
